The Testing Pyramid

DevOps testing isn't just about unit tests; it includes validating infrastructure configuration and security policies before they are applied.

Types of Testing

  • Unit Tests: Testing isolated logic (e.g., Python functions).
  • Integration Tests: Validating interaction between services (e.g., App + Database).
  • Policy Testing: Using tools like 'checkov' or 'tfsec' to scan Terraform for security vulnerabilities.
  • Drift Detection: Verifying that the actual state matches the desired state in Git.
